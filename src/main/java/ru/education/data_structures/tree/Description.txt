Обходить дерево можно двумя путями в ширину и в глубину
Обход в ширину осуществляется с помощью очереди
Обход в глубину - прямой обход(PRE-Order), центрированный обход(IN-Order), обратный порядок(POST-Order)
Можно реализовать с помощью рекурсии
Сбалансированное дерево, дерево в котором левая и правая часть отличается не больше чем на 1 уровень.
