Графы могут быть взвешенными (иметь стоимость прохода по ребру) и невзвешенными.
Также бывают мультиграфы с двумя и более ребрами прохода по одному пути с разнос стоимостью

Для описания графов можно использовать таблицу смежности - представление (матрица) всех графов по вертикали и
горизонтали с описанием смежны они (1) или нет (0) - вариант невзвешенного графа

Для взвешенного графа в таблице прописывам стоимости перехода от одного графа к другому

В данно таблице можно понять есть ли связи в одно и в другую сторону в графах
